# Superior Career Services — Sales Process Flowchart

Interactive sales process flowchart for Superior Career Services & Company LLC, built for David Stevenson's career coaching practice (Tampa Bay, FL).

## Features

- 6-stage interactive pipeline (Outreach → Nurture → Qualify → Consult → Close → Follow-Up)
- Live search across all steps and scripts
- Expandable step cards with detail, decision branches (YES/NO), and coaching tips
- Progress tracker
- Pricing quick-reference (Silver / Gold / Platinum)
- Non-negotiable rules panel

## Tech Stack

- React 19 + Vite
- Pure inline styles (no CSS framework dependency)

## Getting Started

```bash
npm install
npm run dev
```

## Deploy

Configured for Cloudflare Pages:

- **Build command:** `npm run build`
- **Output directory:** `dist`
